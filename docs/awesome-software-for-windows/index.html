Windows 好软件

2021-06-16
468 字

前言

选择依据

软件的选择基本基于以下两条原则：

  1. 开源 > 闭源
  2. 实用性 > 美观度

软件列表

不分类


  • 音乐播放器 YesPlayMusic
    Demo | GitHub
    Feature ( from GitHub ):
    • ✅ 使用 Vue.js 全家桶开发，支持 PWA，支持 Touch Bar
    • 🔴 网易云账号登录（扫码/手机/邮箱登录）
    • 📺 MV 播放、📃 歌词显示、📻 私人 FM / 每日推荐歌曲、☁️ 音乐云盘、🔐 自动音源替换
    • 🤝 功能纯净
    • 🛠 更多特性请访问项目 GitHub 仓库

  • 音乐播放器 LyricEase ( 非开源 )
    Microsoft Store
    Feature:
    • 网易云账号登录
    • Fluent Design、歌词高斯模糊
    • 功能纯净，基础功能完善

  • 泛用型播客客户端 Moon FM ( 非开源 )
    Official
    Feature:
    • 全平台支持 ( iOS / Android / macOS / Windows / Linux )
    • UI 优雅

文本编辑


  • Notepads
    Official | GitHub | Microsoft Store
    Feature:
    • Fluent Design
    • 可完美替代 Windows 自带记事本程序

  • 本地 Markdown 编辑器 Typora ( 非开源 )
    Official
    • 跨平台支持 ( macOS / Windows / Linux )
    • 极度优雅
    • 完全本地
    • 官方提供丰富的主题、支持自定义主题

系统工具

  • 文件管理 Files Official | GitHub | Microsoft Store
    Feature:
    • Fluent Design
    • 支持 QuickLook
    • 完全支持 Windows 资源管理器的右键菜单
    • 简单场景可替代 Windows 资源管理器

  • 音量 / 亮度调整悬浮窗修改 ModernFlyouts
    GitHub | Microsoft Store
    Feature:
    • Fluent Design
    • 无感替代 Windows 原生界面

  • 空格键快速预览 QuickLook
    Official | GitHub | Microsoft Store
    Feature:
    • Fluent Design
    • 插件支持
    • 无感使用

  • 动态壁纸 WinDynamicDesktop:
    GitHub | Microsoft Store
    • 基于PowerShell 的脚本支持（官方脚本网址）
    • 官方提供丰富的动态壁纸
    • 现在有动态壁纸制作工具
