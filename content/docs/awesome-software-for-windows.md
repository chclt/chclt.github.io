## 前言

### 选择依据

软件的选择基本基于以下两条原则：

1. 开源 > 闭源
2. 实用性 > 美观度

## 软件列表

### 不分类

</br>

- 音乐播放器 YesPlayMusic  
  [Demo](https://music.qier222.com/) | [GitHub](https://github.com/qier222/YesPlayMusic)  
  Feature ( from GitHub ):  
  - ✅ 使用 Vue.js 全家桶开发，支持 PWA，支持 Touch Bar
  - 🔴 网易云账号登录（扫码/手机/邮箱登录）
  - 📺  MV 播放、📃 歌词显示、📻 私人 FM / 每日推荐歌曲、☁️ 音乐云盘、🔐 自动音源替换
  - 🤝 功能纯净
  - 🛠 更多特性请访问项目 [GitHub 仓库](https://github.com/qier222/YesPlayMusic)

</br>

- 音乐播放器 LyricEase ( 非开源 )  
  [Microsoft Store](https://www.microsoft.com/store/apps/9n1mkdf0f4gt)  
  Feature:  
  - 网易云账号登录
  - Fluent Design、歌词高斯模糊
  - 功能纯净，基础功能完善

</br>

  - UI  优雅
